# Changelog

## v1.0 (2026-02-13)

Initial release.

### Features
- CANSLIM ratings panel (Composite, RS, EPS, SMR, Acc/Dis)
- Quarterly EPS with YoY % change (via valuewhen change detection)
- Annual EPS + growth rate
- Revenue growth tracking
- EPS Acceleration detection
- Financial metrics (Margin, D/E, Revenue, Net Income)
- 6-point CANSLIM checklist (BUY ZONE / WATCH / AVOID)
- Trend Template (Minervini MA stacking) pass/fail
- S&P 500 overlay (percentage-anchored scaling)
- RS Line (relative strength vs benchmark)
- Moving averages: 21 EMA (pink), 50 DMA (red), 150 DMA (orange), 200 DMA (black)
- EPS surprise markers on chart
- White/light panel theme
- All sections toggleable
- Pine Script v6 compatible
